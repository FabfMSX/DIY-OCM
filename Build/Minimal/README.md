**Components list :** 
- 1x AC415 FPGA board
- 2x 20p double raw female pin header
- 2x 20p double raw male pin header
- 17x 100R resistors
- 1x 22uF capacitors
- 1x 8p dipswitch right angle
- 1x PS/2 connector
- 1x stereo jack connector
- 2x DB9 connectors
- 1x PCB
